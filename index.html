Rpg-maker-resources

A central repository for useful resources related to RPG Maker (VX Ace).

Download this project as a .zip file Download this project as a tar.gz file

What is this?

RPG Maker is an awesome game-making tool by EnterBrain. The core game comes with some resources, and you can extend it in a multitude of ways. Unfortunately, there are two big problems with finding scripts right now:

  1. Link Rot: Sites move or go down. DropBox quotas run dry. You can't always get access to scripts you want.
  2. Versions and Attribution: most scripts are scattered in the RPG Maker forums (or elsewhere on the net)

There is a need for a single, central repository of scripts (just like Ruby's Gems, Java's Maven, or C#'s NuGet). Users can always get access to scripts, and authors can update their scripts freely, giving us a single, always up-to-date place to get access to scripts.

That's what this repository is. Feel free to browse and download whatever you wish. (Future versions may include a local search client.)

How you can help:

  • Add scripts: Send us pull requests to add or update your scripts.
  • Link scripts: Link our home page wherever you're sharing your scripts, so people can find a copy if your copy becomes unavailable.

Master List of Scripts

  • Advanced Game Time: Set and recall the current game time; tints screen to give the appearance of time of day.
  • Auto Loader: Automatically loads all scripts in /Scripts so you can manage and version them like code instead of binary files.
  • Jet Splash Screens: Adds custom splash-screens to your game before the titlescreen.
  • Lemony's Current FPS: Gives you script access to the current in-game FPS.
  • Lemony's Sound-Emitting Events: Create sounds on events that change in volume based on your proximity to them.
  • QFRJ Misc Scripts: Some miscellaneous scripts related to events and party health
  • Vector Engine Light Effects: Field-of-view style lighting. Requires the basic module and light images.
  • Word Wrapping: Automatically wraps messages to fit to the window width.
  • Xail System Title Screen: Customize the title screen to add new menu items, and icons. Requires the XS Core.
  • Xail System Credits: Provides a mechanism for showing a credits screen. Integrates with Xail System Title Screen; requires the XS Core.
